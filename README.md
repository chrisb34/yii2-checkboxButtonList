# yii2-checkboxButtonList

A really simple widget to display a list of checkboxes in button format

## Use

 `
 $weekdays = ['S','M','T','W','T','F','S'];
 CheckboxButtonList::widget([
	 						'name'=>'days',
	 						'selection'=>'',
	 						'items'=>$weekdays
	 		]);
`
